Privacy Policy

SOETIC (Soebandi Ticketing System)
Last Updated: May 21, 2026

1. Introduction

SOETIC (Soebandi Ticketing System) is an internal monitoring dashboard application designed to provide real-time visibility of IT service tickets and operational performance within RSD dr. Soebandi Jember.

This application operates in a strictly read-only mode. Users are only able to view data and are not permitted to create, modify, submit, or delete any data through this application.

2. Information We Collect

SOETIC does not collect any personal data from users’ devices, including but not limited to contacts, location data, media files, or device identifiers.

The application only displays operational data retrieved securely from internal hospital systems, including:

  • Ticket ID, status, priority, and category
  • Issue descriptions submitted through the main system
  • Department or unit information
  • Operational performance metrics (response time, resolution time, assigned personnel)

3. How We Use Information

  • To monitor IT service performance in real time
  • To support operational oversight and management decisions
  • To evaluate service efficiency and response times

4. Data Sharing

SOETIC does not share, sell, rent, or distribute any data to third parties. All data remains within the secure internal infrastructure of the hospital.

5. Data Storage and Retention

The application does not store data locally on user devices. All data is accessed in real time from secure servers. Data retention policies are managed internally by the hospital.

6. Data Security

  • Access is restricted to authorized personnel only
  • Secure authentication mechanisms are implemented
  • All data transmission is encrypted using HTTPS

7. User Rights

Users may access information based on their authorization level. Any discrepancies in displayed data should be reported to system administrators.

8. Children's Privacy

This application is intended for professional use only and is not directed to individuals under the age of 13. We do not knowingly collect data from children.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last Updated" date.
